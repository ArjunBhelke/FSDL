Assignment 1 – Basic GitHub Commands

1. Create a directory
   mkdir FSD-lab

2. Create README file
   echo "Git Lab Experiment" > README.md

3. Navigate into folder
   cd FSD-lab

4. Check git status
   git status

5. Initialize git repository
   git init

6. Add README file
   git add README.md

7. Commit changes
   git commit -m "Initial commit"

8. Add remote repository
   git remote add origin <REPO_URL>

9. Create new branch
   git branch feature-1

10. Rename default branch to main
    git branch -M main

11. Switch to feature branch
    git checkout feature-1

12. Push main branch to GitHub
    git push -u origin main

13. Create new file in feature branch
    echo "This is feature branch work" >> feature.txt

14. Add new file
    git add feature.txt

15. Commit changes
    git commit -m "Added feature.txt"

16. Switch back to main branch
    git checkout main

17. Merge feature branch into main
    git merge feature-1

18. Push feature branch
    git push origin feature-1

19. Push main branch
    git push origin main
